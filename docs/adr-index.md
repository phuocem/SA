# Danh mục Architecture Decision Records (ADR)

Tổng hợp các quyết định kiến trúc quan trọng để tra cứu nhanh.

| ADR | Tiêu đề | Trạng thái |
| --- | ------- | ---------- |
| ADR-001 | Outbox để đảm bảo gửi message tin cậy | Accepted |
| ADR-002 | Phiên bản API theo URI (`/v1`) | Accepted |
| ADR-003 | Idempotency cho consumer qua bảng log tiêu thụ message | Accepted |
| ADR-004 | Saga helper nhẹ cho luồng nhiều bước | Accepted |

## Ghi chú
- Hiện các ADR được ghi ngay trong các hướng dẫn liên quan. Nếu cần bản ADR chi tiết, thêm file `docs/adr/ADR-XXX-title.md` và liên kết vào đây.
